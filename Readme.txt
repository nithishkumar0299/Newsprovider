React News Website

	This project is a React-based web application that fetches news articles from an API and displays them on a user-friendly interface.

Features:

1.Fetches news articles from an external API.
2.Displays news articles with relevant details such as title, author, 3.publication date, and summary.
4.Allows users to filter news articles based on categories.
5.Provides a search functionality to search for specific news articles.
6.Responsive design for seamless viewing on various devices.


Technologies Used:

1.React: A JavaScript library for building user interfaces.
2.Axios: A promise-based HTTP client for making HTTP requests.
3.React Router: A routing library for React applications.
4.Bootstrap: A CSS framework for building responsive and mobile-first websites.
5.News API: An API that provides access to news articles from various sources.

Installation:
1.create react folder:
	npm createvite@latest
2.Navigate to the project directory:
        cd react-news-website
3.Install dependencies:
	npm install
4.Create a .env file in the root directory and add your News API key:
	REACT_APP_NEWS_API_KEY=your_api_key_here
5.Start the development server:
	REACT_APP_NEWS_API_KEY=your_api_key_here
6.Open http://localhost:3000 to view it in the browser

Usage
1.Upon loading the website, you will see a list of news articles fetched from the API.
2.Use the category  to filter articles based on specific categories.
